# metaversegame
# 元宇宙开发者大赛 #

### 1. 项目定位

   基于元宇宙IP的文字冒险NFT游戏
---蛮荒人族
### 2. 项目简介

   【蛮荒人族】是基于超古代神话构建的元宇宙世界。在那里人类将与万族竞争寻求生存。目前我们推出基于此IP的ADV文字冒险游戏和NFT。
    
同时基于NFT和该IP，可以开放构建不同类型的游戏，如RPG、SLG等。
