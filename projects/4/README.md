## 项目简介：

+ 沙盒策略类游戏

+ 用户通过铸造兵种进行入侵，获取随机掉落的装备和稀有兵种

+ 用户可以选择战队状态为防御的用户进行入侵，根据战斗结果对双方进行奖励或者惩罚

+ 用户在游戏期间掉落的装备将记录到智能合约中，并具有唯一ID。
